# Django Chat Application

This is a **Django-based real-time chat application** using **WebSockets**, **Django Channels**, and **Redis**.

---

## Features
- Real-time chat with WebSockets
- Backend APIs with Django REST Framework
- Models, serializers, and views implemented
- Admin panel setup
- JWT authentication for secure API access
- Redis used as channel layer for message routing

---

## Requirements
- Python 3.10+
- Django 5.1.2
- Redis server (running locally on port 6379)
- Git
